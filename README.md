# OTUS_Homeworks

Описание/Пошаговая инструкция выполнения домашнего задания:<br><br>
Повторить создание системы сохранения;<br>
Доделать загрузку данных.<br>

<B>P.S.</B> Я немного добавил от себя, а именно разделил Save и Load на 2 интерфейса, чтобы можно было в каком-то класс реализовать только загрузку или только сохранение, а так-же изменил вариант передачи реализуемых интерфейсов, отдав эту задачу на Zenject (public SaveLoadService(IEnumerable<ISave> saves, IEnumerable<ILoad> loades)). Я оставил возможность добавлять ручками, в случае если объект будет создаваться в рантайме, но заранее известные объекты передаются сразу в конструктор.
