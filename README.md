# OTUS_Homeworks

Описание/Пошаговая инструкция выполнения домашнего задания:<br><br>
Сделать Bootstrap сцену через StateMachine; <br>
Можно начинать через Monobeh или через Zenject; <br>
В конечном варианте бутстрап должен работать через зенжект; <br>
Бутстрап должен создаваться и переходить на игровую сцену, должен отчитаться в консоль, что он создался и прогрелся; <br> 
Создание дополнительных сервисов не обязательно.<br>

<B>P.S.</B> В этот раз я решил упростить State Machine относительно урока. Она умеет переключать состояния, закрывать шторкой, прогревать. Я убрал разделение интерфейсов на IExitableState и IPayloydState, оставив только IState, т.к. передавать данные я могу и другим путем без state machine. Так-же я убрал цикличную зависимость, теперь State не нуждается в State Machine, как на уроке (прокидывали this в конструктор), состояния переключаются извне (в моем случае, в bootstraper). Добавлено событие, уведомляющее о завершении прогрева ресурсов, думал на счет Task, но решил, что с событием будет удобнее работать.
